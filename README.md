# 🏥 HaloMed – Smart Healthcare E-Channeling System

### *Transforming Healthcare Through Digital Appointment Management*

**HaloMed** is a comprehensive web-based **E-Channeling System** developed as a group project for **IT2030 – Object-Oriented Programming** during the **2nd Year, 1st Semester** of the **BSc (Hons) in Information Technology** degree program at **Sri Lanka Institute of Information Technology (SLIIT)**.

The platform aims to modernize traditional healthcare appointment processes by providing patients, doctors, and administrators with a centralized digital solution for appointment scheduling, doctor management, laboratory services, and healthcare administration.

By leveraging modern web technologies and Object-Oriented Programming principles, HaloMed delivers a scalable, secure, and user-friendly healthcare management experience.

---

# 🌟 Project Overview

Healthcare institutions often face challenges such as long waiting times, manual appointment handling, and inefficient record management.

**HaloMed** addresses these challenges by offering a centralized e-channeling platform that enables patients to conveniently book appointments, manage lab tests, and interact with healthcare services online while providing administrators with efficient management capabilities.

The system follows the **MVC (Model–View–Controller)** architectural pattern, ensuring maintainability, scalability, and separation of concerns.

# 🏗️ System Architecture

The application follows the **Model–View–Controller (MVC)** architectural pattern.

### 📦 Model Layer

Responsible for:

* Business logic
* Database interactions
* Data processing
* Application rules

### 🎨 View Layer

Responsible for:

* User interface rendering
* Dynamic web pages using JSP
* User interaction components
* Responsive frontend design

### ⚙️ Controller Layer

Responsible for:

* Request handling
* Application flow management
* Communication between Model and View
* Processing user actions

# 🚀 Getting Started

## Prerequisites

* Java JDK 8 or above
* Apache Tomcat Server
* MySQL Database
* Eclipse IDE / IntelliJ IDEA

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/HaloMed-E-Channeling-System.git
```

2. Import the project into Eclipse or IntelliJ IDEA

3. Configure Apache Tomcat

4. Create the MySQL database

5. Import the provided SQL script

6. Update database credentials

7. Run the application on Tomcat
